异或运算

先看一段PHP代码，它的输出结果是字母a，为什么符号 ! 和 @ 异或结果会变成了字母a。

在PHP进行异或操作的时，会将字符串转化成ASCII值再转化成二进制，然后再进行异或运算（如果两个值相同，异或结果为0。反之则为1），运算完的二进制变成ASCII值最后转成字符串

!的ASCII值为33 二进制为0010 0001
@的ASCII值为64 二进制为0100 0000
异或结果为：0110 0001 对应ASCII值为97 对应字母为a

那么就可以利用这个特性来构造webshell，通过python爆破获得可以异或之后想要的字符

递增/减法

这里引用官方文档：在处理字符变量的算数运算时，PHP 沿袭了 Perl 的习惯，而非 C 的。例如，在 Perl 中 $a = ‘Z’; $a++; 将把 $a 变成’AA’，而在 C 中，a = ‘Z’; a++; 将把 a 变成 ‘[‘（’Z’ 的 ASCII 值是 90，’[‘ 的 ASCII 值是 91）。注意字符变量只能递增，不能递减，并且只支持纯字母（a-z 和 A-Z）。递增／递减其他字符变量则无效，原字符串没有变化。


因为'a'++ => 'b'，只要拿到a就可以通过自增的方式获得a-z的字符，而数组Array刚好有大写A和小写a，意味着已经获得了[a-zA-Z]全部字符了。那要如何获得数组Array，在PHP中如果强制连接和字符串的话，数组将被转化为字符串，其值为Array。那么就可以利用这个特性用来编写webshell。
