[工具]Hydra笔记
发表于 2020-06-19| 更新于 2020-10-13|渗透测试
阅读量:

前言

hydra（九头蛇）是一款非常好用的爆破工具，操作简便而且支持多种协议，图形版名称为：XHydra
Github地址：https://github.com/vanhauser-thc/thc-hydra


参数

参数 说明
-R 继续从上一次进度继续爆破
-S 使用SSL协议连接
-s < port > 指定非默认端口
-l < username > 爆破指定用户名，适合已知用户名时使用
-L < File > 指定用户名字典
-p < password > 爆破指定密码，适合已知密码时使用
-P < File > 指定密码字典
-C < File > 爆破用户密码字典，格式 用户:密码
-t 线程数，默认为16
-w < Time > 超时时间，默认30秒
-o < File > 保存结果到指定文件
-v/V 显示细节
-e < nsr > 可选项 n:空密码试探 s:使用指定用户和密码试探 r:反向登录
-f 成功爆破一个就停止
-M < File > 多目标攻击，格式： 地址:端口
ip service 指定对应IP的对应服务

使用方法

SSH爆破

hydra -L username.txt -P passwd.txt -f ssh://192.168.148.137
用户名字典username.txt，密码字典为passwd.txt来进行爆破

MYSQL爆破

hydra -l root -P passwd.txt -vf 192.168.148.136 mysql
指定用户名root进行爆破，爆破字典为passwd.txt，爆破成即停止

FTP爆破

hydra -L username.txt -P passwd.txt -e ns ftp://192.168.148.137
